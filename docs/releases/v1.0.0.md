# v1.0.0

첫 정식 릴리즈입니다.

## 주요 기능

- `@Route` + `@Reactive` 데코레이터로 REST → live 엔드포인트 전환
- WebSocket / SSE 전송 방식 선택
- MemoryAdapter, PollingAdapter 내장
- 공식 네이티브 어댑터 8종: PostgreSQL, MySQL, MongoDB, Redis, DynamoDB, Elasticsearch, OpenSearch, Snowflake
- 클라이언트 SDK (`routeflow-api/client`) — 자동 재연결, 경로 기반 구독
- Node 내장 `node:sqlite`를 이용한 파일 기반 영구 저장 지원
- 서버 시작 시 `.routeflow/info.json` 자동 생성
- TC39 새 데코레이터 스펙 및 TypeScript 레거시 데코레이터 모두 지원
