# RouteFlow v0.1.0

첫 공개 버전입니다. 이번 릴리스는 `Flux`에서 `RouteFlow`로의 리브랜딩과, RouteFlow의 핵심 메시지를 보여주는 MVP 데모 정리에 초점을 맞춥니다.

## Highlights

- `Flux` -> `RouteFlow` 리브랜딩
- 워크스페이스 패키지명을 `@routeflow/*`로 통일
- 메모리 / PostgreSQL 어댑터 데모를 같은 컨트롤러 코드로 공유
- WebSocket / SSE 전송 레이어를 각각 데모 가능하도록 정리
- 브라우저에서 바로 확인 가능한 데모 페이지 추가

## Included

- `@routeflow/core`
- `@routeflow/client`
- 공식 지원 어댑터
  - `@routeflow/adapter-postgres`
  - `@routeflow/adapter-mysql`
  - `@routeflow/adapter-mongodb`
  - `@routeflow/adapter-redis`
  - `@routeflow/adapter-dynamodb`
  - `@routeflow/adapter-elasticsearch`
  - `@routeflow/adapter-opensearch`
  - `@routeflow/adapter-snowflake`

## MVP Demo

메모리 / Postgres, WebSocket / SSE 조합을 각각 다른 포트로 바로 실행할 수 있습니다.

```bash
pnpm run example:memory
pnpm run example:memory:sse
ROUTEFLOW_POSTGRES_URL=postgresql://user:pass@localhost:5432/routeflow pnpm run example:postgres
ROUTEFLOW_POSTGRES_URL=postgresql://user:pass@localhost:5432/routeflow pnpm run example:postgres:sse
```

기본 포트:

- `3000` - Memory + WebSocket
- `3001` - Memory + SSE
- `3002` - PostgreSQL + WebSocket
- `3003` - PostgreSQL + SSE

## Verification
