# RouteFlow v0.1.1

이번 릴리스는 실제 npm 배포 기준을 정리한 버전입니다.

## Highlights

- 공식 패키지를 `@spear340000/*` 스코프로 공개 배포
- README와 사용 문서를 실제 npm 설치 경로 기준으로 정리
- `docs/`를 RouteFlow 사용 가이드 중심으로 재구성
- `@spear340000/adapter-opensearch`의 타입 패키징 안정화

## Published Packages

- `@spear340000/core`
- `@spear340000/client`
- `@spear340000/adapter-postgres`
- `@spear340000/adapter-mysql`
- `@spear340000/adapter-mongodb`
- `@spear340000/adapter-redis`
- `@spear340000/adapter-elasticsearch`
- `@spear340000/adapter-opensearch`
- `@spear340000/adapter-dynamodb`
- `@spear340000/adapter-snowflake`

## Install

```bash
npm install @spear340000/core @spear340000/client reflect-metadata
```

예시:

```bash
npm install @spear340000/adapter-postgres pg
```

## Verification

- `pnpm run example:build`
- 공식 패키지 개별 빌드 통과
- npm publish 완료

## Notes

- 저장소 이름과 제품명은 계속 `RouteFlow`를 사용한다.
- npm 공개 스코프는 현재 `@spear340000/*`다.
